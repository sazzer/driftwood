# API Features

## Users

### Get User by ID ✓
* Method: GET
* URL: /users/:id

### Update User Profile ✓
* Method: PUT
* URL: /users/:id

### Delete User ✗
* Method: DELETE
* URL: /users/:id

### Search Users ✗
* Method: GET
* URL: /users{...}

## OpenID Clients

### Get Clients for User ✗
* Method: GET
* URL: /clients{?user}

### Get Client by ID ✗
* Method: GET
* URL: /clients/:id

### Create Client ✗
* Method: POST
* URL: /clients

### Update Client ✗
* Method: PUT
* URL: /clients/:id

### Delete Client ✗
* Method: DELETE
* URL: /clients/:id
